.. _ar_programming_mode:

プログラミングモード
==========================

:ref:`ar_play_mode` を体験した後、各機能がどのように実装されているのか、
そして自分のアイデアに合わせて効果を変更したい場合、このプログラミングモードをじっくりと読むことができます。

プログラミングモードの章の各機能は独立したプロジェクトとして作成されており、その数は17あります。
各プロジェクトには対応するコードと説明があり、各機能がどのように動作するかを確認できます。

このプロジェクトはArduinoのプログラミング言語を使用しているため、
始める前にArduino IDEと関連ライブラリをインストールする必要があります。

さあ、発見の旅を始めましょう！
