はじめに
=====================

Zeus Carには、実際に操作するモードとプログラミングモードの2つのモードが設定されています。

* 実際に操作する :ref:`ar_play_mode` とは、Zeus Carを組み立てた後にオンにするだけで、リモコンやアプリを使ってこの車を直接制御し、さまざまなクールな機能を実現できるモードのことです。これは、工場でArduinoボードにコードを事前にアップロードしているためです。

* :ref:`ar_programming_mode` では、Zeus Carのさまざまなクールな機能を体験した後、各プロジェクトごとにそれぞれの機能の実装原理を理解し、それに基づいて自分のアイディアに合わせて変更して、望む機能を実現できます。
